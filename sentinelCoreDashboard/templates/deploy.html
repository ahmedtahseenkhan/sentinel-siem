LINUX One-line install

Run this on any Linux server (Ubuntu/Debian/RHEL/CentOS) as root:

curl -fsSL http://{{ server_ip }}:5050/deploy/install.sh | sudo bash -s -- \
    --server {{ server_ip }} \
    --token {{ enroll_token }} \
    --group production

⬇ Download binary only ⬇ Download install script

Installs as a systemd service with auto-restart. Re-running upgrades the binary in-place.

WINDOWS Install package

Step 1. Download the agent package:

⬇ Download SentinelAgent.zip

Step 2. Extract and run as Administrator:

Expand-Archive SentinelAgent.zip -DestinationPath C:\
cd C:\SentinelAgent
.\install.ps1 -ServerIP {{ server_ip }} -Token "{{ enroll_token }}"

Or remote-deploy to a fleet from a domain admin machine:

.\deploy-to-all-machines.ps1 -ServerIP {{ server_ip }} -CsvFile machines.csv
The installer registers a Windows service with auto-restart on failure (5s/10s/30s backoff).

VERIFY Confirm enrollment

After install, check the agent appeared in the fleet:

curl -H "Authorization: Bearer $WATCHTOWER_API_KEY" \
    http://{{ server_ip }}:9400/api/v1/agents | jq '.data[] | {hostname, status}'

Or just open the main dashboard — the new agent appears within ~30 seconds.
